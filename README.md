# How to use

### Pre-requires:

- ffmpeg
- rye

### Setup dependent packages

`rye sync`

### Run

`rye run python src/tldv_download/tldv.py`



## NOTE
- To download 1hour 10minutes video, This script took 22 minutes. (My Home network is 280Mbps)
